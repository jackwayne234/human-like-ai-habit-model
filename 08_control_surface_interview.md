# Control Surface Interview

Date: 2026-06-27

Purpose: Record a 20-question interview about the human-like AI habit model's mind-control surface, resource body, gate outputs, mode presets, habit automation, and inner/outer attention behavior.

Scope note: The selected answers are intended as default settings for the first model/control-surface version. They are not permanent laws; later versions can tune them through evidence, modes, resource pressure, or user changes.

General design note: The model should allow a state where the mind model is `100%` focused on inner world building while the efficiency model keeps behavior running through automatic deterministic code processes. This is not the constant default, but it should be an available choice. Human sleep is the reference analogy: a large share of life can be spent in an inner-world mode while background body/efficiency systems continue running.

General design note: The purpose of the habit model is to identify repeated useful loops that can be handed off from the mind model to the efficiency model. The efficiency model should take over as much deterministic process work as safely possible, freeing mind-model resources for thinking, imagination, planning, and other higher-level activity.

Rules:

- Ask one question at a time.
- Use multiple choice.
- Mark the recommended answer.
- Record the user's selected answer before moving to the next question.
